# atomic-design-vue-todo-example

## Getting Started

```bash
git clone https://github.com/coder-ka/atomic-design-vue-todo-example

cd atomic-design-vue-todo-example

npm install

npm run dev
```

open [localhost:8080](http://localhost:8080)
